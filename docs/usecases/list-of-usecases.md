GamePulse - Use Cases:
___

### **Gebruiker**
- Login
- Logout
- Register
- Profile instellen (Fully dressed)
- Profile aanpassen
- Wachtwoord veranderen
- Appeal ban

- **Community**
  - Deelnemen
  - Verlaten
  - Eigen aanmaken (Fully dressed)
  - Eigen verwijderen
  - Moderator aanstellen (bij eigen community) (Fully dressed)
- **Post**
  - CRUD (Fully dressed)
- **Chat**
  - CRUD
- Games volgen
- Gebruiker volgen (Fully dressed)
- Overzicht van trending games (al gemaakt in analyse)
- Games zoeken
- Vergelijken van games

- **Wishlist**
  - CRUD

- **Reviews**
  - CRUD (Fully dressed)
  - Like
  - Dislike
  - Rapporteren

- **Andere gebruiker**
  - Blokkeren
  - Rapporteren

___

### **Moderator**
- Community posts deleten

___

### **Administrator**
- Gebruiker blokkeren (Fully dressed)
- Reviews verwijderen
- Community verwijderen
- Community post verwijderen
- Bekijken van rapports (Fully dressed)

___

### **Systeem**
- Aanmaken game
- Aanmaken community
